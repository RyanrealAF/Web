# Forensic Linkage Explorer

**Mission: Reclaiming Narrative Sovereignty through Forensic Documentation.**

This application is a **Non-Linear Knowledge Base**. Documents are not just stored; they are "docked" into a neural web to trace complex social and psychological warfare.

## Core Documentation
- **[Scalability Manifesto](./SCALABILITY_MANIFESTO.md)**: Architecture, aesthetics, and technical constraints.
- **[Methodology](./METHODOLOGY.md)**: Relationality checklists and classification guides.
- **[Handoff Instructions](./HANDOFF_INSTRUCTIONS.md)**: Setup and ingestion workflows.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
